# Credit-Card-Fraud-Detection-Risk-Analytics-
Project Overview

This project focuses on analyzing large-scale credit card transaction data to identify fraud patterns, high-risk merchants, suspicious customer behavior, and fraud-prone transaction categories using MySQL. The project simulates a real-world fraud analytics and risk monitoring system commonly used in banking and fintech industries.

The dataset contains over 1.8 million financial transaction records, making this project suitable for large-scale SQL analytics, KPI reporting, and business intelligence analysis.

Problem Statement

Financial institutions and fintech companies face significant losses due to fraudulent credit card transactions. Detecting suspicious activities across millions of transactions is challenging without proper analytics systems.

The objective of this project is to:

Identify fraud patterns
Detect high-risk merchants and transaction categories
Analyze suspicious customer behavior
Monitor fraud trends across locations and time
Generate business insights for fraud monitoring and risk management
