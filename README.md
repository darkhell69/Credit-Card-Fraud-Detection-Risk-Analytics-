# Credit-Card-Fraud-Detection-Risk-Analytics-
🔍 Project Overview

Built a large-scale Fraud Analytics & Risk Monitoring System using MySQL by analyzing 1.8M+ real-world credit card transactions to uncover fraud patterns, suspicious customer behavior, high-risk merchants, and fraud-prone transaction categories.

This project simulates a real-world banking and fintech fraud monitoring environment used by organizations such as:

Paytm
PhonePe
Barclays
Amazon

The objective was to transform raw transaction data into actionable fraud intelligence using SQL analytics, feature engineering, and business KPI reporting.

📌 Business Problem

Financial institutions process millions of transactions daily, making fraud detection extremely challenging.

Undetected fraud can lead to:

Financial losses
Customer trust issues
Chargeback penalties
Operational risks

This project focuses on identifying:
✅ Fraud patterns
✅ High-risk merchants
✅ Suspicious transaction behavior
✅ Fraud-prone categories
✅ Risk trends across states and timings
